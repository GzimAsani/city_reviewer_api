# City Reviewer Back-end

City Reviewer backend creates necessary API for the frontend



## Built With 

- Rails
- Rspec
- Postgres

### Setup

```bash
# Clone the repo
git clone https://github.com/GzimAsani/city_reviewer_api.git

# cd into the cloned repo
cd track-books-api
```

### Install

```bash
bundle install
```

### Usage

```bash
# Create the database
rails db:create

# Run the migrations
rails db:migrate

# Seed the database with a test account (username: 'test_user', password: '123456)
rails db:seed

# Start the server
rails server
```

### Run tests

```bash
bundle exec rspec
```
